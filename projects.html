Ingredo Tesco Basket

This is an injected popup basket that allows you to add ingredients from a recipe into your Tesco basket without leaving the page. Originally a Backbone project, I was the project lead for a rebuild to stripe out the main dependencies to create a lightweight version while keeping all the functionality.

jQuery, Underscore and Backbone where replaced with vanilla js and a lightweight (2kb) mvc framework named Stapes to cut the size by over 50%.
